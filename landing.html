工作原理

三步操作，三十秒完成。

无需安装。无需注册。你的数据永远不会离开你的设备。

1
打开终端
Mac 用户：按 Cmd + Space，输入 "Terminal"，回车。
Windows 用户：按 Win + R，输入 "cmd"，回车。
2
粘贴以下命令并回车
npx claude-spend

就这么简单。它会临时下载，读取本地 Claude Code 数据，然后启动一个轻量仪表盘。
不会发送任何数据到互联网
3
仪表盘自动在浏览器中打开
新的浏览器标签页会打开，显示你的个人使用仪表盘。清晰看到哪些对话、哪些提示词、哪些日子消耗了最多的额度。
