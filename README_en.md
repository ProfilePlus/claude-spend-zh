## Quick Start

```bash
npx claude-spend-zh
```

Or clone and run locally:

```bash
git clone https://github.com/ProfilePlus/claude-spend-zh.git
cd claude-spend-zh
npm install
npm start
```

Visit **http://localhost:3456**
